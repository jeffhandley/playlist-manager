# Waves & Soul

The biggest hits of classic soul, Motown, and R&B — built around the laid-back, contemplative vibe of "(Sittin' On) The Dock of the Bay." Timeless songs for watching the sunset over the water.

## Playlist Instructions

### Management

- The playlist is defined in this file (`playlists/waves-and-soul.md`) — this is the source of truth
- Use the **playlist-creator** skill for searching and curating new songs to add
- Use the **apple-music-sync** skill for syncing to Apple Music
- For any modifications (adding, removing, reordering):
  1. Update this file first
  2. Sync to Apple Music: `node .github/skills/apple-music-sync/sync.mjs playlists/waves-and-soul.md`

### Ordering

- Songs flow from mellow and reflective to uplifting and warm, with the anchor track up front
- After reordering, renumber all rows sequentially starting from 1
